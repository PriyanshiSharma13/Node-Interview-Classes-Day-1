Q1. What is node.js ?

Node.js is a runtime that runs on chrome's v8 Js engine.

It is a JS running on a server

Q2. What is NPM?

NPM stands for Node Package Manager. It contains all the required modules and packages to run a project. When we install node, we automatically get npm built-in.




Q3. What are the modules in Node.js?

There are 3 modulkes in Node.js

  1. Core modules
  2. Local modules
  3. Third-party modules



Q4. What is the purpose of the module.exports?

module.exports is a method to export a file within node.js and has access to other file by exporting it.

Syntax- const data = { name: "Priyanshi", hobby: "Sketching" } module.exports = data;




Q5. Difference between default export and named export

default export does not have a name where as named export has a name defined within curly braces




Q6. How do you import any module in Node.js
